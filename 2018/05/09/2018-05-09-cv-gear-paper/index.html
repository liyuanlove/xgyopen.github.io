亚像素级边缘检测

背景：边缘较宽
插值法、矩法、拟合法

本文采用的边缘检测方法

本文实现亚像素级齿轮边缘定位的总体思路：
边缘横向方向上对等效像素点灰度值进行插值，
纵向方向上对所得插值像素点坐标值进行拟合。

齿轮几何中心定位方法研究

粗定位：重心法

当齿轮的宽度较小，并且对齿轮图像的质量影响较小时，或齿轮轴线与摄像机光轴轴线相整合重合时，重心法测得的齿轮中心位置是比较可靠的。
当不满足这些情况时，重心法得到的结果可信度较低。

亚像素级定位

Hough变换：不能检测出直径较大且边缘连续性较差的圆形边缘
三点定圆心法：对多组三个非共线的像素点得出的定位结果求取平均值作为最终的定位结果
本文采用：三点法和圆的最小二乘拟合算法（方差小，更准确）

齿轮多参数测量技术（齿轮测量）

齿轮基本几何参数获取

齿轮的基本几何参数：齿数、模数、分度圆直径、齿顶圆直径、齿根圆直径

  1. 齿顶圆、齿根圆直径近似值
    以齿轮的几何中心点为极点，齿轮图像中水平方向为机轴方向，创建极坐标系。极角值为一确定值，极径丛零逐渐增大。
  2. 齿数：连通区域法
    构造圆交点法、连通区域法（P49）
  3. 模数
    模数=分度圆直径（近似值）/齿数（准确值）
    将得到的标准模数值，用标准模数中比较接近的代替
  4. 齿顶圆、齿根圆、分度圆直径的确定
    根据齿数、模数值、用公式计算即可

齿轮主要误差检测（齿距偏差、齿廓偏差、公法线长度变动）

测量实验结果及误差分析

绝对误差在5μm以内

误差分析

  1. 硬件误差
    • 光源不稳定性
    • CCD成像不稳定性
    • CCD分辨率大小
    • 标定板尺寸精度和平整度
  2. 算法误差
    • 几何中心坐标值：只是接近而非真实值
    • 边缘像素的亚像素坐标值：拟合/插值算法计算结果有差异 => 多次运算结果取平均
  3. 环境影响误差
    • 磁场、震动、温度

展望

  1. 渐开线标准直齿圆柱轮 → 已修形齿轮、斜齿轮、锥齿轮
  2. 背光源光照强度、光线颜色 与 边缘定位结果 之间的数学模型

图像预处理技术

图像滤波去噪

本文采用中值滤波：滤除齿轮图像噪声、保持齿轮边缘细节

关于图像滤波（消除噪声、平滑处理）P17

  1. 空域滤波去噪（简单、快速）
    • 线性平滑滤波：领域平均、加权平均
    • 非线性平滑滤波：一维中值滤波、二维中值滤波
  2. 频域滤波去噪

图像二值化处理

本文采用基于阈值分割法的二值化处理：为了能较好的进行后续处理
最佳阈值：用遍历法取得类间方差最大时的阈值（本文中为95）

形态学处理

形态学处理：去除小毛刺、小尖端、不连续间断点引起的空隙（二值化阈值选择引起的）
本文采用开运算

  1. 腐蚀：缩小原图像尺寸
  2. 膨胀：对图像内的小孔、边缘处的小缺口进行填充
  3. 开运算：去除图像边缘的小连接、毛刺、孤立的点，切断图像边缘上的连接，达到平滑效果；低通滤波
  4. 闭运算：去除图像上的小凹陷，填补图像内的小缺口，连接小的间断起到连通作用

边缘检测算法分析

边缘检测：通过图像的梯度来度量
数字图像中，用差分近似梯度

边缘检测算法分为四步：

  1. 滤波
  2. 增强
  3. 检测
  4. 定位

本文选用Canny算子

常用的边缘检测算子（算子比较P28）：

Harris角点检测：为了减少可能存在的伪角点对图像配准的影响，通过角点权值累加取最大值的方法从中选出3个角点，构成三角形。由于图像由二维运动平台平移得到，故三角几何约束关系不变。

  • 模数0.5mm，齿数80，压力角20°
  • 模数0.3mm，齿数133，压力角20°
  • 模数0.8mm，齿数60，压力角20°

流程：

  1. 图像预处理
    CCD工业相机拍摄的局部图像 → 中值滤波 → 去除噪声后的图像

  2. 边缘检测

    • 二值化 → 形态学处理 → 去除毛刺和不连续间断点后的图像
    • Canny算子 → 连续的齿轮边缘
    • 局部图像 → 结合三角几何约束的局部自相关图像拼接算法 → 仿射变换 → 完整齿轮齿廓数据

  3. 齿轮参数计算
    摄像机标定、二维平台俯仰角校准 → 修正后的数据
