*[计算机视觉]CV讨论班

发表于 2018-05-22
研究生师兄弟间组织的、关于计算机视觉的讨论班的笔记整理，主要是读懂论文和复现论文。

讨论班概况

主题

CV: paper + code
代码上尽可能围绕PyTorch展开。

Research方面以CV方向的Paper Reading为主。
Engineering方面以围绕Research做相应展开。
Topic包括但不限于Face，Detection，Segment，Re-ID，pose estimation等方面。

T字形战略

先橫后纵，在保证广度的前提下，发展深度。

CV四个典型任务

  • Classification（分类）
  • Detection（检测）
    • One-Stage：YOLO、SSD
    • Two-Stage：R-CNN系列
  • Segmentation（分割）
    • Semantic
    • Instance
  • Tracking（追踪）
  • 模型压缩与加速

学术界现状

  • Detection：大咖在刚
  • Segmentation+Tracking：可能是我们可以做的方向
  • Classification：已经基本没人做了

Two-Stage的方法之间，尤其是围绕RBG,Kaiming He等人，不同工作相关代码之间耦合度很高。比如，A模型的某块代码是从B模型的代码中拉出来的，所以如果要能把A run过，可能需要对B也可以run通。

或者说关键代码。关注主要矛盾，忽略次要矛盾。哪些是关键？各种群+知乎等，大家讨论或者问的多的，就是关键。很多人不懂，只有读代码才能懂的，我们肯定也需要，这也是我们的优势啊。
