*[拍照]手机拍照及修图技巧

发表于 2018-05-21
一门手机拍照课程的笔记。包含后期修图。

§1 你好，手机摄影！手机摄影必备核心知识大搜罗（手机基础拍照知识）

对焦功能

  1. 什么是对焦
    对焦也叫对光、聚焦。通过相机内的对焦机构，变动物距和相距的位置，使被拍物成像清晰的过程就是对焦。

  2. 如何对焦
    手机对焦：点击屏幕
    日常拍照基本要求：把主体拍清晰（对焦准确）

应用：虚实对比照

应用：虚化光斑拍摄

故意制造虚焦效果：夜晚平常不起眼和杂乱的场景 → 截然不同的虚化感觉

手机虚焦方法一：专业模式虚焦法（适合有专业模式的手机，如OPPO、VIVO、华为、小米等）

  1. 打开手机相机，准备拍摄。
  2. 选择进入相机的专业模式。
  3. 找到焦距选项，横向拨动焦距滑杆，清晰或是模糊我们可以轻松改变。

手机虚焦方法二：全能虚焦法（适合没有专业模式的手机，如iPhone6s等）

  1. 打开相机，这时候画面中的灯都是实焦的。
  2. 让小伙伴把手放到手机镜头前很近的地方，我们点击屏幕对手部进行对焦，然后做一个关键操作，用手指长按屏幕1.5秒左右，会锁定对焦。
  3. 我们把手移开，放大画面就会看到，原来实焦的灯光变虚了。

应用：错位照拍摄

  1. 手机尽量保持在低角度（台阶：从下向上拍）
  2. 设置对焦点在后景人物身上
  3. 积累错位照创意

测光操作

  1. 亮度调节
    手机上下滑动，增减亮度

  2. 大光比环境
    画面亮部暗部对比非常强烈

  3. 应对大光比
    遮挡+道具

  4. 常见光线角度（顺光、逆光、测光、顶光、底光）

①顺光
顺光也叫做“正面光”，指光线的投射方向和拍摄方向相同的光线。顺光时，被摄体受到均匀照明，影调比较柔和。

②逆光
逆光是一种由于被摄主体恰好处于光源和照相机之间的状况，在一般情况下摄影者应尽量避免在逆光条件下拍摄物体，但是逆光情况有时也可用于某些艺术创作。

Tips漂亮的逆光发丝
打开手机闪光灯作为逆光光源，移动到人物的正后方，人物的头发会被打亮

Tips逆光一寸照
逆光+黑白滤镜

应用：逆光剪影照

室外：日落时，让被摄主体背对太阳 → 人物黑，天空亮 → 剪影风格图片

室内：朝阳的窗户，剪影着重强调被摄者的曲线或形态

逆光剪影Tips

1、逆光拍摄，让主体背光
2、日出日落时是理想的拍摄时间
3、被摄者后面的强烈光线，使前景与背景的亮度相差非常大，形成剪影
4、拍摄剪影，着重强调被摄者曲线或形态
5、窗边、山顶、海边、城市 街道
6、手机拍摄剪影只需要在适当环境下，对着背景测光，被摄者就会变黑

③侧光
侧光，即来自景物左侧或右侧的光线，同景物、照相机成90度左右的水平角度。这种光线能产生明显的强烈对比。

“质感照明”：把光源移动到人物测面，测光使被摄主体一测受光，产生强烈的明暗对比，上下滑动屏幕，调节亮度，找到最酷的效果

④顶光
顶光，即来自顶部的光线，与景物、照相机成90度左右的垂直角度。人物在这种光线下，其头顶、前额、鼻头很亮，下眼窝，两腮和鼻子下面完全处于阴影之中，造成一种反常奇特的形态。

因此，一般都避免使用这种光线拍摄人物。(骷髅光)

⑤底光
底光，即来自底部的光线，从下至上与景物、照相机成90度左右的垂直角度。一般底光会有阴森恐怖的感觉。

因为再高的颜值也架不住底光的摧残。

快门使用

  1. 便捷音量键快门
    自拍时：大拇指按下音量键快门

应用：高速慢速快门

  • 高速连拍快门
    拍摄运动物体/抓拍瞬间：按住快门键不放

Tips低视角仰拍跳跃
从低角度拍

Tips低视角仰拍树林
视角仰到与天空平行

  • 慢动作快门
    将原本很快的动作放慢

视频录制

应用：短片拍摄剪辑

视频神器：VUE

VUE拍摄短片的过程：

  1. 同拍照一样，先点击人物对焦
  2. 向左滑动屏幕，挑选一款我们喜欢的滤镜
  3. 引导模特就位，点击开始视频记录。视频拍摄，重在引导和沟通。

Tips朦胧画面技巧
（烟盒的）塑料纸套在手机摄像头上

全景模式

Tips全景分身照

  1. 打开手机的全景模式
  2. 手机缓慢水平移动
  3. 移动到模特时，手机停止，这时模特从摄影师身后绕开
  4. 模特绕到相机位置之外，摆好姿势
  5. 重复以上步骤，继续移动手机，搞定

注意：

  1. 不能让模特直接横着跑过去。
  2. 不要在拍全景的时候跳起来，不然会让人“灵肉分离”。
  3. 全景模式拍大长腿，会把腿拍的更短……
